# PDS Project: Pre-Owned Cars Data Analysis

This project analyzes pre-owned cars data using various data science techniques.

## Features Implemented

- Data cleaning and preprocessing
- Exploratory data analysis with visualizations
- Machine learning model for price prediction
- JSON file manipulation
- Web scraping for additional data
- Multithreading for parallel processing

## Requirements

- Python 3.10+
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn
- requests
- beautifulsoup4

## Usage

Run the `project.ipynb` notebook to see the analysis.

## Dataset

`pre-owned cars.csv` contains data on pre-owned cars including brand, model, price, etc.

## Results

The Random Forest model achieves an R2 score of approximately 0.87 for price prediction.
